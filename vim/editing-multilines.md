If I want to edit a multi lines in a same time, for example like from this:

```
name
comment
phone
email
```

with this:

```
vendor_name
vendor_comment
vendor_phone
vendor_email
```

1. Move the cursor to the `n` in name.
2. Enter visual block mode `ctrlv`.
3. Press `j` three times.
4. Press `I` (capital i).
5. Type in `vendor_`.
6. Press `esc`.
